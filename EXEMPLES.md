## Observations Intéressantes

### Graphes Riches
Certaines entités ont des graphes particulièrement denses :
- **Jazz** → genres, musiciens, lieux, époques
- **Paris** → monuments, personnages historiques, événements
- **Shakespeare** → œuvres, personnages, adaptations

### Connexions Inattendues
Le graphe peut révéler des liens surprenants :
- Quartz → Piezoelectricity → Electronics → Modern technology
- Jazz → New Orleans → French colonization → France
- Jules Verne → Science fiction → Moon landing → Apollo program

### Sources Multiples
Certaines entités ont des données enrichies :
- Musiciens : Wikipedia + MusicBrainz
- Personnages historiques : Wikipedia + Wikidata relations détaillées
- Lieux : Wikipedia + OpenStreetMap (à implémenter)

## Tips

- Commence par des entités simples (1 mot)
- Observe les logs console pour voir les données brutes
- Note quelles entités ont le plus de connexions
- Teste différents types pour voir comment le graphe varie
